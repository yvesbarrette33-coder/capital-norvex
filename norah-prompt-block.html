Bloc Norah — Équipe Norvex

À coller dans le System Prompt de Norah V2 sur ElevenLabs.

Comment l'installer :
1. Clique le bouton « 📋 Copier le bloc » ci-dessus
2. Va dans ElevenLabs → Agents → Norah
3. Dans le grand carré blanc « System Prompt » (qui contient déjà ton prompt v2.0), clique à la toute fin du texte existant
4. Appuie une fois sur Entrée (pour aller à la ligne)
5. Cmd + V pour coller
6. Clique Publier en haut à droite 



✅ Copié dans le presse-papier !
